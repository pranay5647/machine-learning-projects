# Machine Learning Projects

A collection of applied machine learning projects built in Python to demonstrate
end-to-end workflows including data preprocessing, modeling, and evaluation.

## Projects

### 1. Car Evaluation — Random Forest Classification
- Multiclass classification using the UCI Car Evaluation dataset
- Model: Random Forest
- Techniques: encoding categorical features, train/test split, model evaluation
- Metrics: accuracy, precision, recall, F1-score

### 2. Student Performance — Linear Regression
- Predicting student performance using regression
- Model: Linear Regression
- Techniques: feature selection, train/test split, evaluation

## Tech Stack
- Python
- scikit-learn
- pandas
- numpy
- matplotlib

## Purpose
This repository is intended for learning, practice, and portfolio demonstration.
